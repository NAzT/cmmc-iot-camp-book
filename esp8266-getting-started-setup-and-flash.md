# การเริ่มต้นใช้งาน ESP8266 ผ่านทาง Arduino IDE (ตอนที่ 1 – ติดตั้ง Arduino IDE กับ ESP8266 พร้อม flash โปรแกรม)

1. ติดตั้ง Arduino IDE เวอร์ชั้น 1.6.5 หรือ ใหม่กว่า โดย Arduino IDE สามารถ Support  Windows Mac OS X Linux ทั้ง 32bit และ 64 bit ซึ่งหา Download ตัวติดตั้งได้จาก https://www.arduino.cc/en/main/software![](https://i0.wp.com/farm1.staticflickr.com/281/18551638833_b79027b7f3_z.jpg?zoom=2&resize=474%2C173&ssl=1)สำหรับการติดตั้ง Arduino IDE บน Windows จะมีแบบให้เลือกทั้ง แตกไฟล์ใช้ได้เลย (ZIP file for non admin install)
หรือ ติดตั้งเหมือน โปรแกรมทั่วๆไป (Installer) หากติดตั้งเรียบร้อยแล้ว ให้เปิด Arduino IDE ขึ้นมาจะได้หน้าตาแบบนี้เลยครับ![](https://i1.wp.com/farm1.staticflickr.com/285/18984844170_15ffb0b09c_z.jpg?zoom=2&resize=474%2C546&ssl=1)

2. ติดตั้ง Broad ESP8266 ลงบน Arduino IDE

– กดเลือก Menu ไปที่ File >> Preferences
